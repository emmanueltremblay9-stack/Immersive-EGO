# User Guide

Immersive EGO is not gameplay-complete yet. The current build is a Phase 0 technical bootstrap.

## Installing the Current Jar

Install the jar with its required runtime dependencies:

- NeoForge for Minecraft 1.21.1
- Apothic Attributes
- Placebo
- MariesLib
- Cloth Config API on clients

The current jar loads bootstrap code and config only. The survival simulation, `/ego` commands, Apothic mirrored attributes, origins, sleep, nutrition, and recovery systems are planned but not implemented yet.

## Current Config

`enableSimulation` defaults to `false`. Leave it disabled until Phase 1 and the first vertical slice are implemented.

## Expected Gameplay Impact

None yet. This is intentional; unfinished systems must not silently alter gameplay.
