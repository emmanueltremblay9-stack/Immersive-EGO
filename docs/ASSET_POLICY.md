# Asset Policy

- Do not copy textures, icons, sounds, models, GUI atlases, language files, logos, or other assets from Apothic Attributes, Apotheosis, Cold Sweat, Tough As Nails, or any other third-party mod.
- Use only original project assets, generated assets, vanilla references allowed by the Minecraft/NeoForge toolchain, or assets explicitly supplied for Immersive EGO.
- Keep every non-code asset listed in `docs/SOURCE_PROVENANCE.md`.
- Prefer text-only Apothic integration and native Apothic rendering over custom GUI assets.
- Do not add Blockbench models; the mod must not register custom blocks, items, entities, or fluids.

Current project asset:

- `src/main/resources/immersive_ego_logo_1024.png`: supplied Immersive EGO logo.
