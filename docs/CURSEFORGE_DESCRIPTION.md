# CurseForge Description Draft

Immersive EGO is a configurable survival-RPG simulation mod for Minecraft 1.21.1 on NeoForge.

The long-term goal is to connect environment, exposure, hydration, respiration, stamina, sleep, nutrition, pain, psychology, biome origin, acclimation, EGO Load, Readiness, Focus, Comfort, and Recovery Capacity into one deterministic server-authoritative simulation.

Current alpha status: technical bootstrap. The project builds and declares its dependency baseline, but gameplay systems are not complete yet.

Required dependencies:

- Apothic Attributes
- Placebo
- MariesLib

Optional compatibility target:

- Apotheosis

Immersive EGO is not affiliated with or endorsed by those projects.
