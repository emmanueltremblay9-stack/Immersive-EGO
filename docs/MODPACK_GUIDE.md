# Modpack Guide

Current release status: technical bootstrap.

## Required Mods

- NeoForge for Minecraft 1.21.1
- Apothic Attributes
- Placebo
- MariesLib
- Cloth Config API on clients

## Optional Mods

- Apotheosis

## Current Gameplay

No gameplay systems are enabled yet. The bootstrap jar is useful for dependency and load verification only.

## Future Pack Configuration

Pack makers should expect server-side config for gameplay balance and dynamic TOML profile files for climate, origin, food, item weight, synergy, and dimension behavior.
